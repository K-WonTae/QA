# 등록·수납 테이블 (SDL_)

> 출처: `D:\DeskList\업무\학사` 내 신입생 등록금·장학 일괄등록 SQL/TXT 분석
> 공통코드 매핑은 [knowledge/학사/등록](../등록/index.md) 등 기존 코드 문서를 참조한다.

학생 등록금 고지·수납·분납을 관리하는 도메인. 접두어 `SDL_`.

## 코드값 표기 규칙
SQL의 코드 리터럴(예: `'I1601'`)은 공통코드 SubCode와 동일하다. 그룹코드는 앞에 도메인 글자를 붙인다.
`I1601` → 그룹 **SI16**(수납구분)의 `I1601`(완납), `H0801` → **SH08**(차수)의 1차, `Z0201` → **CZ02**(대학구분).
