# 급여구분(A519)

## 업무설명
급여 지급의 종류(급여/강의료/기타급여 등)를 구분한다. AGY_PayMonthDesc의 ProvisionDiv(지급구분) 등에서 사용.

관련업무
- 급여명세(AGY_PayMonthDesc)
- 급여대장(급여/임상급여/강의료 분리 집계)
- 강사료(SSE_Gangsaryo*) 연계

- 3 기타급여
