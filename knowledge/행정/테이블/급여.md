# 급여·호봉 테이블 (AGY_)

> 출처: `D:\DeskList\업무\행정` 내 급여내역·급여대장·보수현황·사대보험/사학연금 쿼리 분석
> 급여 계열 공통코드는 [knowledge/행정/급여](../급여/index.md)에 정리됨 — A501 공제코드 · A502 지급수당 · A506 지급대상 · A512 호봉코드 · A519 급여구분.

교직원 급여 계산·지급·공제를 관리하는 도메인. 접두어 `AGY_`. 키는 `PayYYMM`(급여년월) + `StaffNO`. `StaffMaster`와 사번으로 조인. 함수: `AGY_BonusCalu_Sf`(수당계산), `sf_AGY_최근급여FIND`, `ADM_Pay_GetSubCodeInfo_Sf`.

---

## AGY_PayMonthDesc — 월급여 명세
교직원 월별 급여 명세(지급·공제 집계). 급여대장의 원천.

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| PayYYMM | 급여년월(`YYYYMM`) | |
| StaffNO / StaffName | 사번 / 성명 | |
| OrgID | 소속 | (Organization) |
| JikGubCode | 직급코드 | [직급 A401](../인사/직급(A401).md) |
| JikJongCode | 직종코드 | [직종코드 A402](../인사/직종코드(A402).md) |
| WorkArea | 근무지역 | [근무지역 A400](../인사/근무지역(A400).md) |
| PayRankCode | 급여직급코드 | (AGY_PayRankCode) |
| ProvisionDiv | 지급구분(`1` 급여 / `o` 임상급여 / `4` 강의료 / `9` 계절강의료) | [급여구분 A519](../급여/급여구분(A519).md) |
| PayTotAmt | 지급 총액 | |
| AssignBonusAmt | 보직수당액(지급수당 `A23`) | [지급수당 A502](../급여/지급수당(A502).md) |

> `AGY_PaymonthDesc2015`는 2015년 이전 체계(테이블 분리 운영).

## AGY_PaySalaryStep / AGY_PaySalaryStep2015 — 호봉표(봉급표)
직급·호봉(SS)별 기본 봉급액 기준표. 호봉명 조회: `SF_COR_GongtongCd_Johoe('0','A512'+SS)`.

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| PayRankCode | 급여직급 | |
| SS | 호봉(`01`~`35`, 근가 `41`~`50`, 특 `61`) | [호봉코드 A512](../급여/호봉코드(A512).md) |
| 봉급액 | 본봉(지급수당 `A01` 호봉) | [지급수당 A502](../급여/지급수당(A502).md) |

## AGY_PayRankCode — 급여직급코드
급여 산정용 직급 코드 정의(인사 직급 A401과 매핑).

## AGY_AssignBonusStandard / AGY_ContinuousBonusStandard — 수당 기준
- `AGY_AssignBonusStandard` — 보직수당 기준(보직코드 [A403](../인사/보직코드(A403).md)별 금액)
- `AGY_ContinuousBonusStandard` — 계속(장기)근무수당 기준

## AGY_FixBonusDedu — 고정 상여/공제
기간 적용형 고정 상여·공제 항목.

| 컬럼 | 설명 |
|---|---|
| StaffNO | 사번 |
| StartYYMM / EndYYMM | 적용 시작·종료 년월 |
| BonusDeduDiv | 상여/공제 구분(지급수당 `A23` 보직수당 등) → [지급수당 A502](../급여/지급수당(A502).md) / [공제코드 A501](../급여/공제코드(A501).md) |

## AGY_Dedu* — 공제 항목
4대보험·연금·대출·기타 공제 상세. 공제 항목은 [공제코드 A501](../급여/공제코드(A501).md) 체계와 대응.

| 테이블 | 설명 | 공제코드(A501) |
|---|---|---|
| AGY_DeduKukminPension | 국민연금 공제 | B04 |
| AGY_DeduMedicalInsu | 건강보험 공제 | B03 |
| AGY_DeduEmployment | 고용보험 공제 | B05 |
| AGY_DeduIndustrialAccidentInsu | 산재보험 공제 | |
| AGY_DeduPrivateSchoolPension | 사학연금 공제 | B06 |
| AGY_DeduLoanRepayment | 대출 상환 공제 | B07 연금대부원리금 |
| AGY_DeduStaffDedu | 직원 공제 | B14~B16 기타공제 |
| AGY_DeduEtcDedu | 기타 공제 | B14~B16 기타공제 |

## AGY_PayEtcDesc / AGY_PayException — 급여외 지급 / 급여 예외
- `AGY_PayEtcDesc` — 급여외 기타 지급 명세
- `AGY_PayException` — 급여 예외 처리 대상

## v_AGY_사대보험 / v_AGY_사학연금 — 4대보험 / 사학연금 (뷰)
법정부담금(건강보험_법인·고용보험_법인·산재보험·사학_법인부담금) 집계 뷰. 컬럼: gongje_ym(공제년월), staffno, 건강보험_법인, 고용보험_법인, 산재보험, 법인부담금.

## TF_AGY_PAYCALCBASIC / TF_AGY_PayDate — 급여계산 기초 / 급여일자
급여 계산 배치의 기초 데이터·지급일자 관리.

## AGY_YETA_* / AGY_fromYETA_* — 연말정산(YETA) 연동
연말정산 시스템(YETA) 연동 데이터(`AGY_YETA_UserMatch`, `AGY_fromYETA_CURRENT`, `AGY_fromYETA_RESULT`).

## 기타
- `ais_PayAnnualDesc` — 연봉 명세 / `ais_continuous` — 계속근무
- `MemberDedu` — 회원 공제 / `EtcIncome` — 기타소득
