## 기부 (SGB_)
기부자·기부내역을 관리. 키는 `GiID`(기부자ID, = GibujaID). 명칭 조회: `SF_COR_GongtongCdNm_Johoe('1', GibuGb)`.

### SGB_GiGibon_M — 기부자 기본
기부자(개인/단체) 기본 정보.

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| GiID | 기부자 ID (PK) | |
| Nm / DisNm | 성명 / 표시명 | |
| DaepyoNm | 대표자명(단체) | |
| PhoneNo / DaepyoPhoneNo | 연락처 | |
| ZipNo / Address1 / Address2 | 주소 | |
| (기부단체구분) | 단체 유형 | [기부단체구분 SQ00](../기부/기부단체구분(SQ00).md) |

### SGB_GibuInfo_M — 기부 정보(내역)
기부 1건의 내역(년도·구분·금액·약정).

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| GibujaID | 기부자 ID (→ GiGibon_M.GiID) | |
| Yy | 기부 년도 | |
| GibuGb | 기부구분(`Q1003` 약정 등) | [기부구분 SQ10](../기부/기부구분(SQ10).md) / [기부구분2 SQ11](../기부/기부구분2(SQ11).md) |
| GibuAmt | 기부 금액 | |
| RegisterDt | 등록(기부)일자 | |
| PromiseYn | 약정여부 | |
| ProcessDate | 처리일시 | |

### SGB_GiJigeob_D / SGB_GiGyohoe_D — 직업 / 교회 상세
기부자 부가정보(직업 상세 / 소속 교회). `GiID`로 연결.
