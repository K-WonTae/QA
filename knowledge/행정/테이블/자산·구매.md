# 자산·구매 테이블 (Goods* / Depreciation / EquipPurc*)

> 출처: `D:\DeskList\업무\행정` 내 물품대장·감가상각·폐기리스트·자산일괄입력 쿼리 분석
> 공통코드 매핑은 [knowledge/행정/자산](../자산/index.md)·[구매](../구매/index.md) 참조.

물품·자산의 취득·관리·감가상각·폐기와 구매를 관리하는 도메인. 자산 키는 `AssetNo`(자산코드), 취득 단위는 `AcqNo`(취득번호). 코드명 조회: `SF_COR_GongtongCd_Johoe(0, 'A201'+GoodsDiv)`.

## GoodsMaster — 물품(자산) 대장 마스터
보유 자산 1건. 취득정보(`InstrumentAcq`)·물품코드(`GoodsCode`)·설치장소(`GoodsPart`)와 연결.

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| AssetNo | 자산코드 (PK) | |
| GoodsName | 품명 | |
| GoodsCode | 물품코드 (→ GoodsCode) | |
| Standard / ModelName | 규격 / 모델명 | |
| SetupPlace | 설치장소(= GoodsPart.GoodsPartCode) | |
| AssetStatus | 자산상태(`1` 사용중 등) | [자산상태 A303](../자산/자산상태(A303).md) |
| AcqNo | 취득번호 (→ InstrumentAcq) | |
| LastOrgID | 최종 소속(관리부서) | (Organization) |

## GoodsCode / GoodsMidCode / GoodsPart — 물품코드 / 중분류 / 설치장소
- `GoodsCode` — 물품 분류 코드 마스터
- `GoodsMidCode` — 물품 중분류
- `GoodsPart` — 물품 설치장소(파트) 코드. 명칭: `SF_AJS_GetPartName(SetupPlace)`

## GoodsDisuseMaster / GoodsDisuse — 폐기 마스터 / 상세
자산 폐기(불용) 신청·승인.

| 컬럼 | 설명 |
|---|---|
| RegisterNo | 폐기 등록번호 (마스터·상세 연결키) |
| ReqstDate / ReceiptDate / DisuseDate | 신청일 / 접수일 / 폐기일 |
| AssetNo | 폐기 대상 자산코드 (상세) |

## Depreciation / Depreciation20 / Depreciation90 — 감가상각
자산 감가상각 계산 결과(잔존가치·상각누계). `20`·`90`은 상각방식/내용연수 구분 운영 추정.

| 컬럼 | 설명 |
|---|---|
| AssetNo | 자산코드 |
| Year | 회계연도 |
| 취득가 / 상각액 / 잔존가치 | 감가상각 산출값 |

## GoodsPurcInfo — 물품 구매정보
물품 구매 내역(거래처·금액). 거래처는 `PaymentCompany`/`Customer`와 연계.

## BussCode — 업무(사업) 코드
회계연도·계정구분별 업무(사업) 코드·명칭. 컬럼: AcctDivCode, Year, BussCode, BussName. (회계 도메인과 공유 — [결재·회계](결재·회계.md))
