# 조직·공통 테이블 (Organization / COR_ / BaseCode)

> 출처: `D:\DeskList\업무\행정` 전반의 조인 기준 테이블 분석
> 공통코드 매핑은 [knowledge/행정](../index.md)·[knowledge/공통](../../공통/index.md) 참조.

전 도메인이 공통 조인하는 조직·공통코드 기준 테이블. 학사 도메인의 `COR_` 테이블과 동일 체계를 공유한다([학사/조직·기타](../../학사/테이블/조직·기타.md)).

---

## Organization — 조직(부서)
교직원 소속·기안부서의 조직 단위. 학사 소속코드(`COR_SosogCd_M`)와 `OrgID = SosogCd`로 연결.

| 컬럼 | 설명 |
|---|---|
| OrgID | 조직(소속) ID (PK) |
| OrgName | 조직(부서/학과)명 |

## COR_SosogCd_M — 소속코드 마스터
대학–학부–소속(학과/부서) 조직 코드 기준. 명칭: `SF_COR_SosogNm_Johoe('0', OrgID, 기준일)`.

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| SosogCd | 소속 코드 (PK) | |
| SosogNm | 소속명 | |
| PyeonjeFrDt / PyeonjeToDt | 유효(개편) 기간 | |

## COR_SangwiSosogCd_M — 상위 소속코드
소속의 상위(계층) 매핑. 조직도 트리 구성.

| 컬럼 | 설명 |
|---|---|
| SosogCd / UpSosogCd | 소속 / 상위소속 코드 |
| PrintSeq | 출력 순서 |
| PyeonjeFrDt / PyeonjeToDt | 유효 기간 |

## COR_SosogGyeyeol_M — 소속–계열 매핑
소속별 계열 분류(통계·등록금 계열).

| 컬럼 | 설명 | 공통코드 |
|---|---|---|
| SosogCd | 소속 코드 | |
| GyeyeolGb | 계열구분 | [계열구분 CZ21](../../학사/학사/계열구분(CZ21).md) |
| Gyeyeol5Gb | 계열5구분(통계) | [계열5구분 CZ17](../통계/계열5구분(CZ17).md) |

## COR_TonggyeSosogCd_M — 통계용 소속코드
교육통계 제출용 소속 매핑.

## COR_Date — 날짜(달력)
영업일·학사일정 등 날짜 기준 테이블.

## 공통코드 / 기준정보
| 테이블 | 설명 |
|---|---|
| **BaseCode** | 공통코드 마스터. 컬럼 `GroupCode`, `SubCode`, `CodeName`. 모든 코드 컬럼이 `업무컬럼 = BaseCode.SubCode AND GroupCode='Axxx'`로 조인 |
| Basecode_UseChk | 공통코드 사용여부 체크 |
| CountryCode | 국가코드 ([국가코드 C011](../../공통/국가코드(C011).md)) |
| SchoolInfo | 출신학교 정보 ([학력구분코드 C009](../../공통/학력구분코드(C009).md)) |
| UnivSetting | 대학(시스템) 환경설정 |
| groupuser | 전자결재 권한 그룹(userid, groupid) |

### 코드명 조회 함수
- `SF_COR_GongtongCd_Johoe('0', 그룹코드+세부코드)` — 예: `'A201'+GoodsDiv`
- `SF_COR_GongtongCdNm_Johoe('1', 코드)`
- `ADM_Pay_GetSubCodeInfo_Sf('0', 그룹코드, 코드)` — 급여 계열
